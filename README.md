Welcome to the repository for my personal portfolio website. This project serves as a central hub to showcase my journey as a Second Year Computer Science student at Eduvos, focusing on backend development, networking protocols, and systems programming.
***
🚀 Overview
This portfolio is designed with a "low-level" aesthetic in mind, reflecting my interest in what happens "under the hood" of software. It features a responsive design, dynamic scroll effects, and a glassmorphism UI to highlight my projects and technical skills.
***
Key Features
Dynamic Background Parallax: A custom JavaScript implementation that adjusts the background position based on scroll percentage.
Reveal Animations: Utilizing the Intersection Observer API to animate elements into view as the user scrolls.
Glassmorphism UI: Modern interface design using backdrop-filter and semi-transparent surfaces.
Project Showcase: Integrated links to my work in x86 Assembly and C++.
Contact Integration: A functional contact form powered by Formspree.
***
🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Variables, Flexbox, CSS Grid)
Interactivity: Vanilla JavaScript (ES6+)
Typography: IBM Plex Serif, Work Sans, and Monospace for a "code-centric" feel
Tools: Git/GitHub, Formspree
